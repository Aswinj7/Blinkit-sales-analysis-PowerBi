# Blinkit Sales Analysis Dashboard

## Project Overview

This project analyzes Blinkit's sales performance using Power BI. The goal is to understand how different factors such as item type, fat content, outlet size, and outlet location influence overall sales.

## Tools Used

* Power BI
* Excel / CSV Dataset
* Data Visualization

## Key Metrics

* Total Sales
* Average Sales
* Number of Items
* Average Rating

## Business Questions Solved

1. Total Sales by Fat Content
2. Total Sales by Item Type
3. Fat Content Sales by Outlet
4. Sales Trend by Outlet Establishment Year
5. Sales Distribution by Outlet Size
6. Sales by Outlet Location
7. KPI Comparison by Outlet Type
8. 
## Insights

* Tier 3 outlets generated the highest sales.
* Low fat products contributed more to total sales.
* Supermarket Type 1 had the highest number of items sold.
